# Alterações ao Relatório

## Como usar
- Adiciona cada alteração como um item da lista abaixo
- Indica a secção, página ou parágrafo
- Quando terminares, diz à IA "aplica as alterações"

## Formato sugerido
```
- [Secção/Página]: [o que fazer]
```

## Exemplos
```
- Introdução p.2: reescreve de forma mais formal
- Secção 3.1 último parágrafo: remove
- Conclusão: adiciona mais impacto, está fraca
- Figura 5 legenda: muda para "Arquitetura do sistema"
```
